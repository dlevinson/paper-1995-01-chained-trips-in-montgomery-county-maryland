# Source Data Pointers

The paper states that it uses the 1987-88 Metropolitan Washington Council of Governments home-interview survey. This package does not stage raw household, person, or trip records.

- 1987/88 Metropolitan Washington Council of Governments/National Capital Region Transportation Planning Board Home Interview Survey: https://doi.org/10.15483/1924777
- OSTI dataset record: https://www.osti.gov/biblio/1924777
- Metropolitan Washington Council of Governments Regional Travel Survey page: https://www.mwcog.org/transportation/data-and-tools/household-travel-survey/

The retained local data artifact is an aggregate table workbook derived from the survey. If raw records are ever needed to rerun the FORTRAN workflow, use the public source-data record and document any access terms separately.
